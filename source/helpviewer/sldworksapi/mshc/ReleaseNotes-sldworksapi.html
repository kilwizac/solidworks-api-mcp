Release Notes
 

This topic provides you with quick access to the enhancements in SOLIDWORKS API 2026.

Service Pack 0
  • New functionality
  • New interfaces
  • New delegates, methods, and properties
New functionality
  • Display the Dassault Systemes Physically Based Rendering (DSPBR) appearance visual style. See IDSPBRMaterial, IRenderMaterial::GetDSPBRMaterial, and IRenderMaterial::HasDSPBRProperties.
  • Add family tables to drawings and edit them. See IFamilyTableAnnotation, IFamilyTableFeature, ISheet::InsertFamilyTable.
  • Get top-level items in body folders. See IBodyFolder::GetTopLevelFeatureCount and IBodyFolder::GetTopLevelFeatures.
  • Receive notifications when reordering solid body folders. See DAssemblyDocEvents_SolidBodyFolderReorderNotifyEventHandler and DPartDocEvents_SolidBodyFolderReorderNotifyEventHandler.
  • Get and set the edge thickness when the display style is set to "Shaded with Edges". See ISldWorks::ShadedEdgeThickness.
  • Move parent drawing views independently of child views and receive notifications when changing view positions. See IView::SetViewPosition and DDrawingDocEvents_ViewPositionChangeNotifyEventHandler.
