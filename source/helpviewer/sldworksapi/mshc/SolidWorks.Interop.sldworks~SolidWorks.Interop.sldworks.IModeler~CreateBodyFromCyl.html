Remarks

TheCylArray argument is the following array of doubles:

 [ cylFaceCenter[3], cylAxis[3], cylRadius, cylHeight ]

where:

 

cylFaceCenter[3]

XYZ location that represents the center of one of the cylinder faces

cylAxis[3]

XYZ direction; the cylinder is extruded along this vector from the cylFaceCenter location for a distance of cylHeight

cylRadius

Cylinder radius

cylHeight

Length to extrude along the cylAxis direction; if cylHeight is 0, then a sheet body is created of dimension cylRadius and whose normal is defined by cylAxis
Requirements

Target Platforms: Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
