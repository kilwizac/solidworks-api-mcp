Remarks

To set the text format, you must be editing the sketch. See IModelDoc2::EditSketch or ISketchManager::InsertSketch.

To change the text format of an existing sketch text, get the ITextFormat object using ISketchText::GetTextFormat or ISketchText::IGetTextFormat, next use the various ITextFormat APIs to set the text format properties to what you want the sketch text to look like. Then use this method to change how the sketch text looks. To make the sketch text use the default model text format, use this method with the useDoc argument set to True.

To determine if this sketch text is using the default model text format or not, use ISketchText::GetUseDocTextFormat.
