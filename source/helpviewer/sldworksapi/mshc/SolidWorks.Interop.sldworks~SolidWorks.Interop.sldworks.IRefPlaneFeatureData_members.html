Type

Gets the type of reference plane created in SOLIDWORKS 2009 or earlier. Can also get whether a constraint-based reference plane created in SOLIDWORKS 2010 or has angle or offset distance references.

 NOTE: This property is a get-only property. Set is not implemented. 

 
Type2

Gets whether the reference plane is constraint based; thus, created in SOLIDWORKS 2010 and later.

NOTE: This property is a get-only property. Set is not implemented. 

 
UpdatePlane Gets or sets whether to update this reference plane so that it is parallel to the screen.  
UseNormalPlane

Gets or sets whether to: 

  • Use the plane normal to the selected plane

  • Automatically size the plane to either the geometry on which it is created or to the bounding box of the model geometry
