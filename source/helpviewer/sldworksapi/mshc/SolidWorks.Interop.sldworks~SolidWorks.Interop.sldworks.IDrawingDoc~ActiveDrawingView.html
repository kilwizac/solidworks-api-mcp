property System.Object^ ActiveDrawingView {
   System.Object^ get();
}

Property Value

Currently active drawing view
Remarks

This property returns the active drawing view, if one is active. If the drawing sheet is active, the property returns NULL and you can use IDrawingDoc::GetCurrentSheet to get the current sheet for this drawing.

This is a read-only property. Use IDrawingDoc::ActivateView to set the active drawing view.
