Remarks

A width mate constrains a tab between two planar faces.

IMateFeatureData is the parent of this advanced mate interface.

To create a width mate:

  1. Follow general instructions in the IAssemblyDoc::CreateMate Remarks.
  2. Specify IWidthMateFeatureData::TabSelection or use IModelDocExtension::SelectByID2 to pre-select the tab using Mark = 16.
  3. Specify IWidthMateFeatureData::WidthSelection or use IModelDocExtension::SelectByID2 to pre-select the width using Mark = 1.
  4. Specify IWidthMateFeatureData::ConstraintType.
  5. Specify other properties of the WidthMateFeatureData object as required.

To edit a width mate:

  1. Access its feature and call IFeature::GetDefinition to get the MateFeatureData object.
  2. Cast the MateFeatureData object to a WidthMateFeatureData object.
  3. Modify the WidthMateFeatureData object.
  4. Call IFeature::ModifyDefinition.

To delete a width mate, use IModelDocExtension::DeleteSelection2.
