Remarks

The index for both rows and columns is 0-based.

Tables and cells:

  • BOM table annotation: All cells can be edited.
  • General table annotation: All cells can be edited.
  • Hole table annotation: Only some columns can be edited because some information is automatically generated, thus, that information cannot be edited. The header row and custom columns can be edited.
  • Revision table annotation: All cells can be edited.
Requirements

Target Platforms: Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
