Remarks

This method trims the curve from the start point to the end point in a parametric direction. It returns NULL if the curve is discontinued or non-linear between the points.

If your application calls ISurface::CreateTrimmedSheet4 or ISurface::ICreateTrimmedSheet4 to create a sheet with a boundary defined by arcs (IModeler::CreateArc or IModeler::ICreateArc) or lines (IModeler::CreateLine or IModeler::ICreateLine), then your application must first call this method to trim the arc and line curves before passing them to ISurface::CreateTrimmedSheet4 or ISurface::ICreateTrimmedSheet4.

If your application calls ISurface::CreateTrimmedSheet4 or ISurface::ICreateTrimmedSheet4 to create a sheet with an elliptical boundary (IModeler::CreateEllipse or IModeler::ICreateEllipse), then you do not need to trim the elliptical curve before passing it to ISurface::CreateTrimmedSheet4 or ISurface::ICreateTrimmedSheet4.
