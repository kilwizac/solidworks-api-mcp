Remarks

Before using this method, select the base entity to act as the datum point for the ordinate dimension and any additional entities to include in the group of ordinate dimensions.

Selections made immediately after calling this method continue to add ordinate dimensions to the group of ordinate dimensions. When you finish adding ordinate dimensions to the group, use IModelDoc2::SetPickMode to return to the default selection mode.

Use IModelDoc2::EditOrdinate to add ordinate dimensions to an existing group of ordinate dimensions.
