Remarks

An angle mate places selected items at a specified angle. A limit angle mate is an advanced mate that allows items to move within a range of angles. Use IAngleMateFeatureData::IsAdvancedMate to determine whether this is a limit angle mate.

IMateFeatureData is the parent of this mate interface.

To create either an angle mate or a limit angle mate:

  1. Follow general instructions in the IAssemblyDoc::CreateMate Remarks.
  2. Specify IAngleMateFeatureData::EntitiesToMate or use IModelDocExtension::SelectByID2 to pre-select the entities using Mark = 1.
  3. Specify IAngleMateFeatureData::ReferenceEntity or use IModelDocExtension::SelectByID2 to pre-select the reference entity using Mark = 67108864.
  4. Specify other properties of the AngleMateFeatureData object.

To edit an angle mate:

  1. Access its feature and call IFeature::GetDefinition to get the MateFeatureData object.
  2. Cast the MateFeatureData object to an AngleMateFeatureData object.
  3. Modify the AngleMateFeatureData object.
  4. Call IFeature::ModifyDefinition.

To delete this angle mate, use IModelDocExtension::DeleteSelection2.
