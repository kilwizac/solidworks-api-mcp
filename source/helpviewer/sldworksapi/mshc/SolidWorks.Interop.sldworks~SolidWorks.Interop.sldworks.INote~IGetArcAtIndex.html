Remarks

Call INote::GetArcCount before calling this method to determine the size of the return array.

Return value is an array of doubles:

 [ LineType, StartPt[3], EndPt[3], CenterPt[3], RotDir ]

where :

LineType

Line type. Valid returns are defined in swLineTypes_e. A LineType is a combination of a line style and line weight.

StartPt[3]

Array of 3 doubles (X,Y,Z) describing the start point.

EndPt[3]

Array of 3 doubles (X,Y,Z) describing the end point. If the arc is closed, then this is the same point as the StartPt.

CenterPt[3]

Array of 3 doubles (X,Y,Z) describing the center point.

RotDir

Rotational direction (CW = -1, CCW = 1)

The data returned from this method is in terms of document apace.
