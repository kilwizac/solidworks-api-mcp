Remarks

To successfully use this method, you must first turn on 3D Interconnect by either:

  • Setting Tools > Options > System Options > Import > Enable 3D Interconnect

    - or -

  • Calling ISldWorks::SetUserPreferenceToggle(swUserPreferenceToggle_e.swMultiCAD_Enable3DInterconnect, True)

To edit the imported assembly, save the SOLIDWORKS assembly and then use IFeature::GetImportedFeatureParameters and IFeature::SetImportedFeatureParameters.

See the IImport3DInterconnectData Remarks.

If 3D Interconnect is turned off, use IAssemblyDoc::AddComponent5 instead of this method.

Requirements

Target Platforms: Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
