SolidWorks.Interop.sldworks Namespace : IAssemblyDoc Interface

For a list of all members of this type, see IAssemblyDoc members.

Public Properties
  Name Description
ClearanceVerificationManager Gets the clearance verification manager.  
EnableAssemblyRebuild Gets or sets whether to suspend the rebuild of the assembly.  
EnablePresentation Gets or sets whether the assembly is in presentation mode.  
InterferenceDetectionManager Gets the Interference Detection manager, which allows you to run interference detection on an assembly to determine whether components interfere with each other.  
IsSupportedMatesAvailable Gets whether this assembly contains mates that are supportive of a mate controller.  
LargeDesignReviewTransparencyLevel Gets or sets the transparency level of unmodified components in the graphics area of this assembly opened in Large Design Review mode.  
LargeDesignReviewTransparencyLevelDynamic Gets or sets whether to dynamically modify the transparency level of unmodified components in the graphics area when the transparency level slider is moved on the Filter Modified Components PropertyManager page.  
LargeDesignReviewTransparencyLevelEnabled Gets or sets whether transparency levels are activated for unmodified components in the graphics area for this assembly opened in Large Design Review mode.
