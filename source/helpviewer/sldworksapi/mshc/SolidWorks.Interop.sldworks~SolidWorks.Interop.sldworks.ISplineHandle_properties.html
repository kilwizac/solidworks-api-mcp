ISplineHandle Interface Properties
   

SolidWorks.Interop.sldworks Namespace : ISplineHandle Interface

For a list of all members of this type, see ISplineHandle members.

Public Properties
  Name Description
Curvature Gets or sets the degree of curvature at any point where curvature control was added.  
CurvatureControlled Gets or sets whether the spline handle has curvature control.  
Editable Gets or sets whether this spline handle can be edited.  
Parameter Gets value in the parameter space of the curve underlying the spline handle.  
RadiusOfCurvature Gets or sets the radius of curvature at any spline point.  
SplinePointNumber Gets the number of the points of this spline handle.  
TangentDriving Enables or disables spline control using tangent magnitude, tangent radial direction, and tangent polar direction.  
TangentMagnitude Gets or sets the weight for the tangency magnitude in the specified direction for this spline handle.  
TangentPolarDirection Gets or sets the tangent polar direction, which is the elevation angle of the tangent vector to a plane placed at a point perpendicular to a spline point.  
TangentRadialDirection Gets or sets the tangent radial direction, which controls the tangency direction by allowing modification of the spline's angle of inclination relative to the x, y, or z axis.  
X Gets or sets the x coordinate of the spline point.  
Y Gets or sets the y coordinate of the spline point.  
Z Gets or sets the z coordinate of the spline point.
