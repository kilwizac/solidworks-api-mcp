Remarks

This method creates a point in the active 2D or 3D sketch. If a sketch is not active, the point is not created and NULL is returned. Use IModelDoc2::GetActiveSketch2 or IModelDoc2::IGetActiveSketch2 to check to see if the sketch is active.

IModelDoc2::SetAddToDB and IModelDoc2::SetDisplayWhenAdded increase performance during entity creation by adding entities directly to the SOLIDWORKS database. IModelDoc2::SetAddToDB also avoids inferencing.

Requirements

Target Platforms: Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
