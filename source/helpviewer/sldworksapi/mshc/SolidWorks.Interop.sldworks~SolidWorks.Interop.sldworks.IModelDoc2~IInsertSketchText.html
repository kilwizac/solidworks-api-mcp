Remarks

Alignment and FlipDirection are valid only when a curve, edge, or sketch segment is selected. Text appears along the selected entity. If an entity is not selected, Text appears horizontally starting at the origin. Select the curve, edge, or sketch segment with Mark = 1.

See the SOLIDWORKS Design user-interface help > Sketching > Sketch Entities > Sketch Text > SketchText PropertyManager topic for more information about this functionality.
