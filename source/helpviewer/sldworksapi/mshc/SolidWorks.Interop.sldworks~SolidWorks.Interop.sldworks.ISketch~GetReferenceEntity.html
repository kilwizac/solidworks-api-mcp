Parameters

LEntityType

Entity type as defined in swSelectType_e (only swSelFACES and swSelDATUMPLANES are valid values for this method)

Return Value

Either a reference plane or a face, depending on the value of lEntityType
Remarks
If the sketch resides on a face that is consumed by subsequent features, then this method returns NULL and swSelNOTHING. To get to the face, edit the sketch using IModelDoc2::EditSketch.
