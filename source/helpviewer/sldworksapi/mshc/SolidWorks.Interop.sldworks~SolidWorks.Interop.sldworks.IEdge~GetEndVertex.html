Remarks

If no vertex exists for this edge (such as the edge of a newly created cylinder), then this method returns NULL.

This method and IEdge::GetStartVertex or IEdge::IGetStartVertex return distinct vertices, unless the edge is closed. Because edges have no natural direction, you cannot necessarily predict which of the two points you will get first and which last.

Use ICoEdge::GetCurveParams or ICoEdge::IGetCurveParams to get consistent start and end positions.
