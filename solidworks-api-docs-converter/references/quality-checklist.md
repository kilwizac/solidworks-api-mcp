# Quality Checklist

- Every method has a complete signature.
- Every parameter has a type and description.
- Every enum value has a numeric value documented.
- All code examples have language tags.
- No broken internal links.
- Frontmatter validates against the schema.
- Keywords include common search terms.
- Related members are linked bidirectionally where possible.
- Categories are from the approved taxonomy.
- Version information is accurate or set to "unknown".
