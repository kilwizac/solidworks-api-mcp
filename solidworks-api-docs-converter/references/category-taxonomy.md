# Category Taxonomy

Use only these category ids in `categories` arrays.

- core: Fundamental interfaces (ISldWorks, IModelDoc2)
- documents: Document operations
- features: Feature creation and manipulation
- sketches: Sketch operations
- selections: Selection management
- geometry: Bodies, faces, edges, vertices
- dimensions: Dimensions and annotations
- drawings: Drawing-specific operations
- assemblies: Assembly-specific operations
- parts: Part-specific operations
- file-io: Import and export operations
- ui: User interface interactions
- constants: Enumerations and constant groups
- errors: Error handling and HRESULT patterns
