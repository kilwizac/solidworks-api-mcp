# swWitnessLineVisibility_e

Values for the visibility of extension lines.

## Values

| Member | Value | Description |
|---|---|---|
| `swWitnessLineBoth` | 0 | = Both extension lines are visible |
| `swWitnessLineFirst` | 1 | = Only the first extension line is visible |
| `swWitnessLineNone` | 3 | = Neither extension line is visible |
| `swWitnessLineSecond` | 2 | = Only second extension line is visible |
