# swSlotMateConstraintOptions_e

Slot mate constraint options.

## Values

| Member | Value | Description |
|---|---|---|
| `swSlotMateConstraintOption_Centered` | 1 | = Center the component in the slot |
| `swSlotMateConstraintOption_Distance` | 2 | = Place the component axis at a specified distance from the end of the slot |
| `swSlotMateConstraintOption_Free` | 0 | = Allow the component to move freely in the slot |
| `swSlotMateConstraintOption_Percent` | 3 | = Place the component axis at a specified percent of slot length distance from the end of the slot |
