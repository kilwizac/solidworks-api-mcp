# swDrawingSheetsZonesLetterLayout_e

Options for letter layouts in drawing sheet zones.

## Values

| Member | Value | Description |
|---|---|---|
| `swDrawingSheetsZonesLetterLayout_Column` | 1 | = Letters appear in columns from the origin and numbers appear in rows |
| `swDrawingSheetsZonesLetterLayout_Row` | 0 | = Letters appear in rows from the origin and numbers appear in columns |
