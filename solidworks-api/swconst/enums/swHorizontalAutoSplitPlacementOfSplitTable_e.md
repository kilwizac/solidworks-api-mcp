# swHorizontalAutoSplitPlacementOfSplitTable_e

Options for placement of the split table portion after a horizontal table split.

## Values

| Member | Value | Description |
|---|---|---|
| `swHorizontalAutoSplitPlacementOfSplitTable_BelowLastSplit` | 1 | = Place the bottom split portion below the top split portion |
| `swHorizontalAutoSplitPlacementOfSplitTable_NextToLastSplit` | 0 | = Place the bottom split portion to the right of the top split portion |
