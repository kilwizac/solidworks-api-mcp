# swCosmeticWeldBeadSide_e

Cosmetic weld bead options.

## Values

| Member | Value | Description |
|---|---|---|
| `swCosmeticWeldBeadSide_allAround` | 3 | = Apply the weld bead to the selected face or edge and to all adjacent faces or edges |
| `swCosmeticWeldBeadSide_bothSides` | 2 | = Apply the weld bead to the selected face or edge and to the opposite face or edge |
| `swCosmeticWeldBeadSide_selection` | 1 | = Apply the weld bead to the selected face or edge |
