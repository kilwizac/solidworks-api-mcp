# swPropertyManagerPageCursors_e

PropertyManager page's cursor after a user makes a selection in the SOLIDWORKS graphics area.

## Values

| Member | Value | Description |
|---|---|---|
| `swPropertyManagerPageCursors_Advance` | 2 | = Move to the next selection box on the PropertyManager page |
| `swPropertyManagerPageCursors_None` | 0 | = None |
| `swPropertyManagerPageCursors_Okay` | 1 | = Okay and close the PropertyManager page |
