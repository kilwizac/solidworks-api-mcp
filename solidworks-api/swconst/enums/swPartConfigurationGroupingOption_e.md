# swPartConfigurationGroupingOption_e

Part configuration groupings for BOM table display.

## Values

| Member | Value | Description |
|---|---|---|
| `swDisplay_AllConfigurationOfSamePart_AsOneItem` | 2 | = display configurations of the same part as one item |
| `swDisplay_ConfigurationOfSamePart_AsSeparateItem` | 1 | = display configurations of the same part as separate items |
| `swDisplay_ConfigurationWithSameName_AsOneItem` | 3 | = display configurations with the same name as one item |
