# swTabSlotFeatureSpacingType_e

Types of spacing between tabs/slots in Tab and Slot groups.

## Values

| Member | Value | Description |
|---|---|---|
| `EqualSpacing` | 0 | = Specify the number of instances of tab/slot pairs |
| `SpacingLength` | 1 | = Specify a distance between tabs/slots; the number of instances of tab/slot pairs is calculated internally |
