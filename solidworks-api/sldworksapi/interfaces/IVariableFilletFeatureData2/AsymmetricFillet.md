# IVariableFilletFeatureData2.AsymmetricFillet

Gets or sets whether this variable radius fillet is asymmetric.

## Signature

```csharp
System.Boolean AsymmetricFillet {get; set;}
```
## Parameters

None.

## Return Value

True if an asymmetric fillet, false if a symmetric fillet
