# ISketchLine.GetBendLineDirection

Gets whether the sketch line is a bendline, and, if it is, the direction of the bendline.

## Signature

```csharp
System.Int32 GetBendLineDirection()
```
## Parameters

None.

## Return Value

Bendline direction as defined in swBendLineDirection_e
