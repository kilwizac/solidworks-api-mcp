# IAppearanceSetting.Reflection

Gets or sets the reflectivity of a surface.

## Signature

```csharp
System.Double Reflection {get; set;}
```
## Parameters

None.

## Return Value

0 <= reflectivity <= 1, where 0 is for no reflection, and 1 is for a mirror reflection
