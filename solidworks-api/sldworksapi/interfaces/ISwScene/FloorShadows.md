# ISwScene.FloorShadows

Gets or sets whether to show shadows cast by the model on the scene floor.

## Signature

```csharp
System.Boolean FloorShadows {get; set;}
```
## Parameters

None.

## Return Value

True to show shadows cast by the model, false to not
