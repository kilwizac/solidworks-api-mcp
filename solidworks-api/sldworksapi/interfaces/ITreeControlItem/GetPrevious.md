# ITreeControlItem.GetPrevious

Gets the previous sibling of this item in the FeatureManager design tree.

## Signature

```csharp
TreeControlItem GetPrevious()
```
## Parameters

None.

## Return Value

Previous sibling of this item in the FeatureManager design tree

## Remarks

Use
ITreeControlItem::GetNext
to get the next sibling of this item in the FeatureManager design tree.
