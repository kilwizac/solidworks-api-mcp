# ITreeControlItem.GetRoot

Gets the root item of the FeatureManager design tree.

## Signature

```csharp
TreeControlItem GetRoot()
```
## Parameters

None.

## Return Value

Root item of the FeatureManager design tree

## Remarks

Use
ITreeControlItem::IsRoot
to get whether an item is the root item of the FeatureManager design tree.
