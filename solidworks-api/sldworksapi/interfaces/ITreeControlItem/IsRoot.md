# ITreeControlItem.IsRoot

Gets whether this item is the root item of the FeatureManager design tree.

## Signature

```csharp
System.Boolean IsRoot {get;}
```
## Parameters

None.

## Return Value

True if this item is the root item of the FeatureManager design tree, false if not

## Remarks

Use
ITreeControlItem::GetRoot
to get the root item of the FeatureManager design tree.
