# IDrSection.ScaleHatchPattern

Gets or sets whether to scale the hatch pattern to the section view.

## Signature

```csharp
System.Boolean ScaleHatchPattern {get; set;}
```
## Parameters

None.

## Return Value

True to scale the hatch pattern to the section view, false to not
