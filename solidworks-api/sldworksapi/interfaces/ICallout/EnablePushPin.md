# ICallout.EnablePushPin

Gets or sets whether to enable the pushpin for this callout.

## Signature

```csharp
System.Boolean EnablePushPin {get; set;}
```
## Parameters

None.

## Return Value

True to enable the callout pushpin, false to disable it
