# ISweepFeatureData.Path

Gets or sets the sweep path for this sweep feature.

## Signature

```csharp
System.Object Path {get; set;}
```
## Parameters

None.

## Return Value

A set of sketched curves contained in one sketch, a curve, or a set of model edges
