# IMate2.CanBeFlipped

Gets whether this distance or angle mate can be flipped.

## Signature

```csharp
System.Boolean CanBeFlipped {get;}
```
## Parameters

None.

## Return Value

True if mate can be flipped, false if not

## Remarks

Use
IMate2::Type
to determine the type of mate.
