# IModelDoc2.GetRayIntersectionsTopology

Gets the topology intersections generated by IModelDoc2::RayIntersections.

## Signature

```csharp
System.Object GetRayIntersectionsTopology()
```
## Parameters

None.

## Return Value

Array of containing the hit points (see Remarks )

## Remarks

This method only returns information if the swRayPtsOptsTOPOLS option was specified in the Options argument of
IModelDoc2::RayIntersections
.
The array returned contains information about each entity hit of each ray on the given bodies. The topology hit can be coordinated with the hit point using the BodyIndex return value from
IModelDoc2::GetRayIntersectionsPoints
. You can determine the type of item hit (face, edge or vertex) by checking the IntersectionType return value in the IModelDoc2::GetRayIntersectionsPoints.
