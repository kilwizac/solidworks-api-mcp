# IModelDoc2.IGetRayIntersectionsTopology

Gets the topology intersections generated by IModelDoc2::IRayIntersections.

## Signature

```csharp
System.Object IGetRayIntersectionsTopology()
```
## Parameters

None.

## Return Value

in-process, unmanaged C++: Pointer to an array containing the hit entities (see Remarks ) VBA, VB.NET, C#, and C++/CLI: Not supported See In-process Methods for details about this type of method.

## Remarks

This method only returns information if the swRayPtsOptsTOPOLS option was specified in the Options argument of
IModelDoc2::RayIntersections
.
The array returned contains information about each entity hit of each ray on the given bodies. The topology hit can be coordinated with the hit point using the BodyIndex return value from
IModelDoc2::IGetRayIntersectionsPoints
. You can determine the type of item hit (face, edge or vertex) by checking the IntersectionType return value in the IModelDoc2::IGetRayIntersectionsPoints.
