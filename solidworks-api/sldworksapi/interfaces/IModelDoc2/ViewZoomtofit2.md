# IModelDoc2.ViewZoomtofit2

Zooms the currently active view to fit the screen.

## Signature

```csharp
void ViewZoomtofit2()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

This method differs from IModelDoc2::ViewZoomtofit method only in that it zooms in closer to the geometry and has less of a border.
