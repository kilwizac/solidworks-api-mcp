# IModelDoc2.ClosePrintPreview

Closes the currently displayed Print Preview page for this document.

## Signature

```csharp
void ClosePrintPreview()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

Use this method after using
IModelDoc2::PrintPreview
to close a Print Preview page.
