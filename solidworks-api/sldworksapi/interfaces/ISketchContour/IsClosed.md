# ISketchContour.IsClosed

Gets whether this sketch contour is open or closed.

## Signature

```csharp
System.Boolean IsClosed()
```
## Parameters

None.

## Return Value

True if the sketch contour is open, false if closed

## Remarks

This method works even if the sketch is suppressed.
