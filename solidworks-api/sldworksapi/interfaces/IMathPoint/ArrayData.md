# IMathPoint.ArrayData

Gets or sets the array of x,y,z coordinates that describe this math point.

## Signature

```csharp
System.Object ArrayData {get; set;}
```
## Parameters

None.

## Return Value

Array of 3 doubles representing the x,y,z coordinates of the math point
