# IMathPoint.IArrayData

Gets or sets the array of x,y,z coordinates that describe this math point.

## Signature

```csharp
System.Double IArrayData {get; set;}
```
## Parameters

None.

## Return Value

in-process, unmanaged C++: Pointer to an array of 3 doubles representing the x,y,z coordinates of the math point VBA, VB.NET, C#, and C++/CLI: Not supported
