# ICenterOfMass.IGetCoordinates

Gets the coordinates of this center of mass.

## Signature

```csharp
System.Double IGetCoordinates()
```
## Parameters

None.

## Return Value

in-process, unmanaged C++: Pointer to an array of doubles of the x, y, and z coordinates of the center of mass VBA, VB.NET, C#, and C++/CLI: Not supported See In-process Methods for details about this type of method.
