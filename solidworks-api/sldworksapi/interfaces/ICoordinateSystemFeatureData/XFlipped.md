# ICoordinateSystemFeatureData.XFlipped

Gets or sets whether to flip the x axis of this coordinate system feature.

## Signature

```csharp
System.Boolean XFlipped {get; set;}
```
## Parameters

None.

## Return Value

True to flip the x axis, false to not
