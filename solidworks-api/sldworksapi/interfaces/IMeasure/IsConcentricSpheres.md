# IMeasure.IsConcentricSpheres

Gets whether the two selected entities are concentric spheres.

## Signature

```csharp
System.Boolean IsConcentricSpheres {get;}
```
## Parameters

None.

## Return Value

True if the two selected entities are concentric spheres, false if not
