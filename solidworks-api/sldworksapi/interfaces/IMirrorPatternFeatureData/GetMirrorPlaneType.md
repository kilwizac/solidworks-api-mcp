# IMirrorPatternFeatureData.GetMirrorPlaneType

Gets whether the mirror plane is a face or a reference plane.

## Signature

```csharp
System.Int32 GetMirrorPlaneType()
```
## Parameters

None.

## Return Value

Type of object specified as the mirror plane: 0 = face 1 = plane
