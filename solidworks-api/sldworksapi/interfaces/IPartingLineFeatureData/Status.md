# IPartingLineFeatureData.Status

Gets the status of this parting line feature.

## Signature

```csharp
System.Int32 Status()
```
## Parameters

None.

## Return Value

Status of parting line feature as defined in swPartingLineFeatureStatus_e

## Remarks

Call this method after specifying the settings and options for this feature to find out the status of the feature.
