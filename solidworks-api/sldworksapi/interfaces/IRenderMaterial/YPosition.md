# IRenderMaterial.YPosition

Gets or sets the Y offset direction for the appearance.

## Signature

```csharp
System.Double YPosition {get; set;}
```
## Parameters

None.

## Return Value

Y offset direction

## Remarks

This property offsets the appearance from its original position.
Call
IRenderMaterial::XPosition
to set the X coordinate for the appearance.
