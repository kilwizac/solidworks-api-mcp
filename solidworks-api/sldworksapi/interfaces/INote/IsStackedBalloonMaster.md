# INote.IsStackedBalloonMaster

Gets whether this note is the master note of a balloon stack.

## Signature

```csharp
System.Boolean IsStackedBalloonMaster()
```
## Parameters

None.

## Return Value

True if this note is the master note of a balloon stack, false if not
