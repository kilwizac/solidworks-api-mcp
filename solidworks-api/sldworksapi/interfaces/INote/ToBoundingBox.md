# INote.ToBoundingBox

Gets and sets whether to select the To bounding box option for leaders in this note's PropertyManager page.

## Signature

```csharp
System.Boolean ToBoundingBox {get; set;}
```
## Parameters

None.

## Return Value

True to select the To bounding box option, false to not

See the SOLIDWORKS Help for information about the
To bounding box
option.
