# INote.IsCompoundNote

Obsolete for documents created in SOLIDWORKS 2016 and later. Gets whether the current note is a compound note.

## Signature

```csharp
System.Boolean IsCompoundNote()
```
## Parameters

None.

## Return Value

True if note is compound, false if not
