# ISurfExtrudeFeatureData.D1DraftOn

Gets or sets whether to draft the extruded surface in Direction 1.

## Signature

```csharp
System.Boolean D1DraftOn {get; set;}
```
## Parameters

None.

## Return Value

True to draft the extruded surface in Direction 1, false to not
