# ISurfExtrudeFeatureData.D1CapEnd

Gets or sets whether to cap the end of the extruded surface in Direction 1.

## Signature

```csharp
System.Boolean D1CapEnd {get; set;}
```
## Parameters

None.

## Return Value

True to cap the end in Direction 1, false to not
