# ISurfExtrudeFeatureData.BothDirections

Gets or sets whether this surface is extruded in both directions.

## Signature

```csharp
System.Boolean BothDirections {get; set;}
```
## Parameters

None.

## Return Value

True extrudes the surface in both directions, false does not
