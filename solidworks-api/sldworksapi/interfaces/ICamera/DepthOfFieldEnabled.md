# ICamera.DepthOfFieldEnabled

Gets whether depth of field effects are enabled or disabled.

## Signature

```csharp
System.Boolean DepthOfFieldEnabled {get; set;}
```
## Parameters

None.

## Return Value

True if depth of field effects are enabled, false if they are disabled
