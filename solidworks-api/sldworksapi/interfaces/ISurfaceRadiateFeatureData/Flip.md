# ISurfaceRadiateFeatureData.Flip

Gets or sets whether to flip the direction of this surface radiate feature.

## Signature

```csharp
System.Boolean Flip {get; set;}
```
## Parameters

None.

## Return Value

True flips the radiate direction, false does not
