# IEdgeFlangeFeatureData.ReverseOffset

Gets or sets whether to flip the flange length for this edge flange.

## Signature

```csharp
System.Boolean ReverseOffset {get; set;}
```
## Parameters

None.

## Return Value

True flips the flange length, false does not (default)
