# ISheet.TitleBlock

Gets the title block in this sheet.

## Signature

```csharp
TitleBlock TitleBlock {get;}
```
## Parameters

None.

## Return Value

Title block in this sheet

## Remarks

Only one title block can exist in a sheet. This property gets that title block, if it exists.
