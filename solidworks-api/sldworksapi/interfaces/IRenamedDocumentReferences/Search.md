# IRenamedDocumentReferences.Search

Searches for documents whose references to update.

## Signature

```csharp
void Search()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

This method is only available if
IRenamedDocumentReferences::UpdateWhereUsedReferences
is true.
