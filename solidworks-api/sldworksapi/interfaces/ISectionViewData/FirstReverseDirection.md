# ISectionViewData.FirstReverseDirection

Gets or sets whether to reverse the first direction of the section view for this part or assembly.

## Signature

```csharp
System.Boolean FirstReverseDirection {get; set;}
```
## Parameters

None.

## Return Value

True to reverse the first direction of the section view, false to not EndOLEPropertyRow
