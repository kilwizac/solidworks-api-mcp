# ISketchPatternFeatureData.PropagateVisualProperty

Gets or sets whether to propagate visual properties (i.e., colors to all pattern instances).

## Signature

```csharp
System.Boolean PropagateVisualProperty {get; set;}
```
## Parameters

None.

## Return Value

True to propagate visual properties, false to not
