# ISketchPatternFeatureData.UseCentroid

Gets or sets whether to use a centroid for this sketch pattern feature.

## Signature

```csharp
System.Boolean UseCentroid {get; set;}
```
## Parameters

None.

## Return Value

True to a centroid, false to use a reference point
