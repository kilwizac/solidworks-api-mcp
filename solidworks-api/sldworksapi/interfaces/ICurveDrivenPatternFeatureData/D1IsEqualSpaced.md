# ICurveDrivenPatternFeatureData.D1IsEqualSpaced

Gets or sets whether the pattern items are equally spaced in Direction 1.

## Signature

```csharp
System.Boolean D1IsEqualSpaced {get; set;}
```
## Parameters

None.

## Return Value

True spaces the pattern items equally, false does not
