# ICurveDrivenPatternFeatureData.D1ReverseDirection

Gets or sets whether the pattern direction is reversed in Direction 1.

## Signature

```csharp
System.Boolean D1ReverseDirection {get; set;}
```
## Parameters

None.

## Return Value

True reverses the pattern direction, false does not
