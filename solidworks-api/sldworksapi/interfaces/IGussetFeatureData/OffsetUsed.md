# IGussetFeatureData.OffsetUsed

Gets or sets whether offset is used to locate the profile for this gusset feature.

## Signature

```csharp
System.Boolean OffsetUsed {get; set;}
```
## Parameters

None.

## Return Value

True if offset is used, false if not
