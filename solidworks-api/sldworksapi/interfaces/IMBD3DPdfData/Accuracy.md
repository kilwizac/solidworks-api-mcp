# IMBD3DPdfData.Accuracy

Gets or sets the level of accuracy for lossy compression when publishing to SOLIDWORKS MBD 3D PDF.

## Signature

```csharp
System.Int32 Accuracy {get; set;}
```
## Parameters

None.

## Return Value

Level of accuracy for lossy compression as defined in sw3DPDFAccuracy_e
