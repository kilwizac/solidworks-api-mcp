# ISimpleCornerTreatmentFeatureData.CornerTreatmentTrimType

Gets and sets the corner treatment trim type for this simple corner treatment feature.

## Signature

```csharp
System.Int32 CornerTreatmentTrimType {get; set;}
```
## Parameters

None.

## Return Value

Corner treatment trim type as defined by swCornerTreatmentTrimType_e

## Remarks

This property is valid only for swCornerTreatmentTrimType_e.:
swCornerTreatmentTrim_BodyTrim
swCornerTreatmentTrim_PlanarTrim
