# IChainPatternFeatureData.PathPlaneReference

Gets or sets the reference plane for the path selected for this chain pattern feature.

## Signature

```csharp
System.Object PathPlaneReference {get; set;}
```
## Parameters

None.

## Return Value

Reference plane

## Remarks

This property is:
only available when the path is a
sketch line
.
available for all types of chain patterns.
See
Accessing Selections that Define Features
for additional details on using this property.
