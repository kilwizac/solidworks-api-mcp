# IChainPatternFeatureData.Group2PathLink1

Gets or sets Path Link 1 in Chain Group 2 in this chain pattern feature.

## Signature

```csharp
System.Object Group2PathLink1 {get; set;}
```
## Parameters

None.

## Return Value

Path Link 1 in Chain Group 2 : Cylindrical face , Circular or linear edge , Sketch point , Vertex , or Reference axis

## Remarks

This property is only available for connected linkage chain patterns.
