# IChainPatternFeatureData.Group1PathLink2

Gets or sets Path Link 2 in Chain Group 1 in this chain pattern feature.

## Signature

```csharp
System.Object Group1PathLink2 {get; set;}
```
## Parameters

None.

## Return Value

Path Link 2 in Chain Group 1 : Cylindrical face , Circular or linear edge , Sketch point , Vertex , or Reference axis

## Remarks

This property is only available for distance linkage and connected linkage chain patterns.
See
Accessing Selections that Define Features
for additional details on using this property.
