# IDSPBRMaterial.EmissionMode

Gets or sets the emission mode for this material.

## Signature

```csharp
System.Int32 EmissionMode {get; set;}
```
## Parameters

None.

## Return Value

0 = luminous emittance (default) or 1 = luminous power
