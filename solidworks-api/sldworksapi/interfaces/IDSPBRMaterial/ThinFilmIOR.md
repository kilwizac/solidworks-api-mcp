# IDSPBRMaterial.ThinFilmIOR

Gets or sets the thin film layer index of refraction for this material.

## Signature

```csharp
System.Double ThinFilmIOR {get; set;}
```
## Parameters

None.

## Return Value

1.0 <= Thin film layer index of refraction <= 2.5; 1.3 is the default
