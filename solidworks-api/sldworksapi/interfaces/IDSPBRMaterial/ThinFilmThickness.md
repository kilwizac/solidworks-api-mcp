# IDSPBRMaterial.ThinFilmThickness

Gets or sets the thin film layer thickness for this material.

## Signature

```csharp
System.Double ThinFilmThickness {get; set;}
```
## Parameters

None.

## Return Value

0.0 <= Thin film layer thickness in microns <= 1.0; 0.4 is the default
