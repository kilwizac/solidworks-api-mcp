# IStructuralMemberFeatureData.GetPathSegmentAt

Gets the sketch segment associated with the body of this structural member.

## Signature

```csharp
SketchSegment GetPathSegmentAt( 
   Body2
PBodyIn
)
```
## Parameters

- `PBodyIn` (Body2): Body

## Return Value

Sketch segment

## Remarks

The body generated by this feature is not available when the feature is rolled back. Therefore, it is not necessary to call
IStructuralMemberFeatureData::AccessSelections
or
IStructuralMemberFeatureData::ReleaseSelectionAccess
before or after calling this method.
