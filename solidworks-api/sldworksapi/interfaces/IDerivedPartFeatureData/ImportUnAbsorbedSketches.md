# IDerivedPartFeatureData.ImportUnAbsorbedSketches

Gets or sets whether to import unabsorbed sketches with the derived part feature.

## Signature

```csharp
System.Boolean ImportUnAbsorbedSketches {get; set;}
```
## Parameters

None.

## Return Value

True to import unabsorbed sketches, false to not
