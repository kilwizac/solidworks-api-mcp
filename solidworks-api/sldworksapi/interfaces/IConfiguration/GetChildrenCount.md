# IConfiguration.GetChildrenCount

Gets the number of children for this configuration.

## Signature

```csharp
System.Int32 GetChildrenCount()
```
## Parameters

None.

## Return Value

Number of children in this configuration

## Remarks

Call this method before calling
IConfiguration::IGetChildren
.
