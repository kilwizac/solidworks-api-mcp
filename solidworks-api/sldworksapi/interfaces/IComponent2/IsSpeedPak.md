# IComponent2.IsSpeedPak

Gets whether the active configuration for this component is SpeedPak.

## Signature

```csharp
System.Boolean IsSpeedPak {get;}
```
## Parameters

None.

## Return Value

True if the active configuration for this component is SpeedPak, false if not
