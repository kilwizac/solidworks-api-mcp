# IJogFeatureData.UseDefaultBendRadius

Gets or sets whether to use the default bend radius for this jog feature.

## Signature

```csharp
System.Boolean UseDefaultBendRadius {get; set;}
```
## Parameters

None.

## Return Value

True uses the default bend radius, false does not
