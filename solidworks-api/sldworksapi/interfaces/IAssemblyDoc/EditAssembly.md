# IAssemblyDoc.EditAssembly

Switches back to the assembly document for editing.

## Signature

```csharp
void EditAssembly()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

This method is used with
IAssemblyDoc::EditPart2
, which allows you to edit a part within the context of an assembly.
