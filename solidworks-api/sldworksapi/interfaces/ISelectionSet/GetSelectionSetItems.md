# ISelectionSet.GetSelectionSetItems

Gets the selection set items in this selection set.

## Signature

```csharp
System.Object GetSelectionSetItems()
```
## Parameters

None.

## Return Value

Array of selection set items

## Remarks

To get the types of entities in the selection set, call
ISelectionSet::GetSelectionSetItemTypes
.
