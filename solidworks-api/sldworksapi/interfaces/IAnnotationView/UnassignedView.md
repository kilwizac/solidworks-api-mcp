# IAnnotationView.UnassignedView

Gets whether this annotation view is assigned to a 3D View.

## Signature

```csharp
System.Boolean UnassignedView {get;}
```
## Parameters

None.

## Return Value

True if this annotation view is not assigned to a 3D View, false if it is
