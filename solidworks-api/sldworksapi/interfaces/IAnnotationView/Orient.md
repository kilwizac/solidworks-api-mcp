# IAnnotationView.Orient

Orients this annotation view.

## Signature

```csharp
System.Boolean Orient()
```
## Parameters

None.

## Return Value

True if the annotation view is oriented, false if not

## Remarks

The annotation view is oriented normal to its plane.
