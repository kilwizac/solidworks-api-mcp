# IMiterFlangeFeatureData.UsePositionTrimSideBends

Gets or sets whether to remove extra material in neighboring bends for this miter flange feature.

```csharp
System.Boolean UsePositionTrimSideBends {get; set;}
```
## Parameters

None.

## Return Value
