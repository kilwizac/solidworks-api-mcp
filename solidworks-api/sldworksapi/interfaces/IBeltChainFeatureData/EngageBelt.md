# IBeltChainFeatureData.EngageBelt

Gets and sets whether to engage the belt with the pulleys.

## Signature

```csharp
System.Boolean EngageBelt {get; set;}
```
## Parameters

None.

## Return Value

True to engage the belt with the pulleys, false to suppress belt mates
