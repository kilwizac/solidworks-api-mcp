# IDisplayDimension.Jogged

Gets or sets whether this ordinate or angular running dimension is jogged.

## Signature

```csharp
System.Boolean Jogged {get; set;}
```
## Parameters

None.

## Return Value

True if dimension is jogged, false if it is not
