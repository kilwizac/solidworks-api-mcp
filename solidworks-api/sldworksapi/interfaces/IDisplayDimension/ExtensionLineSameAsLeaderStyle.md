# IDisplayDimension.ExtensionLineSameAsLeaderStyle

Gets or sets whether to use leader line styles for extension line styles.

## Signature

```csharp
System.Boolean ExtensionLineSameAsLeaderStyle {get; set;}
```
## Parameters

None.

## Return Value

True to use leader line styles for extension line styles, false to not
