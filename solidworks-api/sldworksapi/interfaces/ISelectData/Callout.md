# ISelectData.Callout

Gets or sets the callout to attach to the selected object.

## Signature

```csharp
Callout Callout {get; set;}
```
## Parameters

None.

## Return Value

Callout to which to attach the selected object

## Remarks

This property is not used by
IModelDocExtension::MultiSelect
or
IModelDocExtension::IMultiSelect
.
