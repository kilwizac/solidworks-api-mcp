# ISketchPicture.Angle

Gets or sets the rotation angle of the picture in the sketch.

## Signature

```csharp
System.Double Angle {get; set;}
```
## Parameters

None.

## Return Value

Rotation angle in radians of the picture; counter-clockwise from the positive X
