# ILocalLinearPatternFeatureData.D1TotalInstances

Gets or sets the total number of instances in Direction 1, including skipped items, in this linear component pattern feature.

## Signature

```csharp
System.Int32 D1TotalInstances {get; set;}
```
## Parameters

None.

## Return Value
