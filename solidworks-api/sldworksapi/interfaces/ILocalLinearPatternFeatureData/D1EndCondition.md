# ILocalLinearPatternFeatureData.D1EndCondition

Gets or sets how to specify the spacing of pattern instances in Direction 1 of this linear component pattern feature.

## Signature

```csharp
System.Int32 D1EndCondition {get; set;}
```
## Parameters

None.

## Return Value

Spacing of pattern instances in Direction 1 as defined in swPatternEndCondition_e
