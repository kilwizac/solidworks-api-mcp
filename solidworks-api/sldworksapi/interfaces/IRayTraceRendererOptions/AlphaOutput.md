# IRayTraceRendererOptions.AlphaOutput

Gets or sets whether to render the model as alpha or final color output.

## Signature

```csharp
System.Boolean AlphaOutput {get; set;}
```
## Parameters

None.

## Return Value

True for alpha output, false for final color output
