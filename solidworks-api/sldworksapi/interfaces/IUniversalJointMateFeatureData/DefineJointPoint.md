# IUniversalJointMateFeatureData.DefineJointPoint

Gets or sets whether to define a joint point.

## Signature

```csharp
System.Boolean DefineJointPoint {get; set;}
```
## Parameters

None.

## Return Value

True to define a joint point, false to not

## Remarks

If this property is set to true, then specify
IUniversalJointMateFeatureData::JointPoint
.
