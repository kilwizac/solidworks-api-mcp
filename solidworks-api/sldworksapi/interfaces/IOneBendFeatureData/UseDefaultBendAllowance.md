# IOneBendFeatureData.UseDefaultBendAllowance

Gets or sets whether to use the default bend allowance for this bend.

## Signature

```csharp
System.Boolean UseDefaultBendAllowance {get; set;}
```
## Parameters

None.

## Return Value

True to use the default bend allowance, false to not
