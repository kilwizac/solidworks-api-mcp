# IDetailCircle.GetView

Gets the the drawing view that displays this detail circle.

## Signature

```csharp
View GetView()
```
## Parameters

None.

## Return Value

Pointer to the IView object

## Remarks

This is the owning drawing view from which the detail view was derived.
