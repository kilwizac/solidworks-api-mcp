# IThickenFeatureData.Merge

Gets or sets whether to merge the results of this thicken feature in a multibody part.

## Signature

```csharp
System.Boolean Merge {get; set;}
```
## Parameters

None.

## Return Value

True enables the merge results option, false disables it
