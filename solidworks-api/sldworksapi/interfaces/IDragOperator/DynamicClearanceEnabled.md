# IDragOperator.DynamicClearanceEnabled

Gets or sets the dynamic clearance setting.

## Signature

```csharp
System.Boolean DynamicClearanceEnabled {get; set;}
```
## Parameters

None.

## Return Value

True enables the dynamic clearance option, false disables it
