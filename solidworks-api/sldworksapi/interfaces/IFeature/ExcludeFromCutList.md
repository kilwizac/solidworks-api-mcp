# IFeature.ExcludeFromCutList

Gets or sets whether to exclude this feature from the cut list.

## Signature

```csharp
System.Boolean ExcludeFromCutList {get; set;}
```
## Parameters

None.

## Return Value

True to exclude, false to not

## Remarks

This method only works for cut list items.
