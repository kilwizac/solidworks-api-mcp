# ILocalCurvePatternFeatureData.D1AlignmentMethod

Gets or sets how to align the instances in this curve-driven component pattern feature.

## Signature

```csharp
System.Int32 D1AlignmentMethod {get; set;}
```
## Parameters

None.

## Return Value

Alignment as defined in swLocalCurvePatternAlignment_e
