# ISplineHandle.Reset

Resets the selected handle to its initial state on this spline handle.

## Signature

```csharp
void Reset()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

Use
ISketchSpline::ResetAllHandles
to reset all spline handles to their initial state.
