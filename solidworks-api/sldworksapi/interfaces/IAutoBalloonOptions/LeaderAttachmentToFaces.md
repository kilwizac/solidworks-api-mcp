# IAutoBalloonOptions.LeaderAttachmentToFaces

Gets and sets whether to attach balloons to faces.

## Signature

```csharp
System.Boolean LeaderAttachmentToFaces {get; set;}
```
## Parameters

None.

## Return Value

True to attach balloons to faces; false to attach balloons to edges
