# IAutoBalloonOptions.CustomSize

Gets and sets the user-defined size of the balloons.

## Signature

```csharp
System.Double CustomSize {get; set;}
```
## Parameters

None.

## Return Value

User-defined size of the balloons; valid only when IAutoBalloonOptions::Size is set to swBalloonFit_e .swBF_UserDef
