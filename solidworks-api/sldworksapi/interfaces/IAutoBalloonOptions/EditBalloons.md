# IAutoBalloonOptions.EditBalloons

Gets and sets whether to use edit balloon options when creating the auto balloon.

## Signature

```csharp
System.Boolean EditBalloons {get; set;}
```
## Parameters

None.

## Return Value

True to use edit balloon option as specified in IAutoBalloonOption::EditBalloonOption , false to not
