# IHelixFeatureData.ReverseDirection

Gets or sets whether to make this helix feature extend backward from the point of origin.

## Signature

```csharp
System.Boolean ReverseDirection {get; set;}
```
## Parameters

None.

## Return Value

True to extend backward from the point of origin, false to not

## Remarks

For...
Reverses direction by...
Helixes
Extending the helix backwards from the point of origin
Spirals
Creating an inward spiral
