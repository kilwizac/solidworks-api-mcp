# IPropertyManagerPageCheckbox.State

Gets or sets the state of this check box.

## Signature

```csharp
System.Int32 State {get; set;}
```
## Parameters

None.

## Return Value

State of this check box as defined in swPropertyManagerCheckboxState_e

## Remarks

If you use this property to set the state of a check box, then you must use this property to modify the state of the check box.
