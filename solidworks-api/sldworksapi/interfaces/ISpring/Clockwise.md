# ISpring.Clockwise

Gets or sets the direction of the coils of the spring.

## Signature

```csharp
System.Boolean Clockwise {get; set;}
```
## Parameters

None.

## Return Value

True to set the direction of the coils of the spring clockwise, false to set their direction counter-clockwise
