# ISpring.TaperAngle

Gets or sets the angle by which to taper a compression spring.

## Signature

```csharp
System.Double TaperAngle {get; set;}
```
## Parameters

None.

## Return Value

Angle by which to taper the spring

## Remarks

Use
ISpring::TaperOutward
to set the direction of the taper outward or inward.
