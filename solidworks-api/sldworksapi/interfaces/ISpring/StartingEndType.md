# ISpring.StartingEndType

Gets or sets the starting end type for either an extension spring or a torsion spring.

## Signature

```csharp
System.Int32 StartingEndType {get; set;}
```
## Parameters

None.

## Return Value

Starting end type as defined in swSpringExtensionEndType_e for an extension spring or swSpringTorsionEndType_e for a torsion spring
