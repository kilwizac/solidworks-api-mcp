# IDocumentSpecification.DocumentType

Gets or sets the type of model document to open.

## Signature

```csharp
System.Int32 DocumentType {get; set;}
```
## Parameters

None.

## Return Value

Type of document as defined by swDocumentTypes_e EndOLEPropertyRow

## Remarks

This property is not valid for opening
3D
EXPERIENCE files using
SOLIDWORKS Design Augmented
.
