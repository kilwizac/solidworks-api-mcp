# IDocumentSpecification.UseLightWeightDefault

Gets or sets whether to use the system default lightweight setting.

## Signature

```csharp
System.Boolean UseLightWeightDefault {get; set;}
```
## Parameters

None.

## Return Value

True to use the system default lightweight setting, false to use IDocumentSpecification::Lightweight value
