# IDrawingDoc.GetSheetNames

Gets a list of the names of the drawing sheets in this drawing.

## Signature

```csharp
System.Object GetSheetNames()
```
## Parameters

None.

## Return Value

Array containing the names of the drawing sheets in this drawing
