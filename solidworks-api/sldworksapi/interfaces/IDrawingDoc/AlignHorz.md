# IDrawingDoc.AlignHorz

Uses the selected edge to align the current drawing view.

## Signature

```csharp
void AlignHorz()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

This method aligns the drawing view so that the selected edge is horizontal.
