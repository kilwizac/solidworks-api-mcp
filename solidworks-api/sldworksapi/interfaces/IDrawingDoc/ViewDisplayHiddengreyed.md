# IDrawingDoc.ViewDisplayHiddengreyed

Sets the current display mode to Hidden Lines Visible.

## Signature

```csharp
void ViewDisplayHiddengreyed()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

The hidden lines are shown in grey.
