# ITabAndSlotGroupData.TabThickness

Gets or sets the tab thickness.

## Signature

```csharp
System.Double TabThickness {get; set;}
```
## Parameters

None.

## Return Value

Tab thickness

## Remarks

This property defines the thickness of
ITabAndSlotGroupData::TabFace
, which is a planar face shared with
ITabAndSlotGroupData::SelectionTabEdge
.
