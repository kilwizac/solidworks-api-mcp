# IThreadFeatureData.InitializeThreadData

Initializes this thread feature's data with default values.

```csharp
void InitializeThreadData()
```
## Parameters

None.

## Return Value

Unknown.

This method initializes all of the properties of this interface to the default values documented for each property.
