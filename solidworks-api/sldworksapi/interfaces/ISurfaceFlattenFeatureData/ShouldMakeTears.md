# ISurfaceFlattenFeatureData.ShouldMakeTears

Gets or sets whether to enable relief cuts for this surface-flatten feature.

## Signature

```csharp
System.Boolean ShouldMakeTears {get; set;}
```
## Parameters

None.

## Return Value

True to enable relief cuts, false to not

## Remarks

Tear edges
identify where to make the relief cuts before flattening the face or surface.
