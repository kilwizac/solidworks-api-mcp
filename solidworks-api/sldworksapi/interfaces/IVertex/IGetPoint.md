# IVertex.IGetPoint

Gets the point corresponding to this vertex.

## Signature

```csharp
System.Double IGetPoint()
```
## Parameters

None.

## Return Value

in-process, unmanaged C++: Pointer to an array of three doubles representing the x, y, and z coordinates of the point VBA, VB.NET, C#, and C++/CLI: Not supported See In-process Methods for details about this type of method.
