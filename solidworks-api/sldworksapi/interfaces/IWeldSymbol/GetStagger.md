# IWeldSymbol.GetStagger

Gets whether this weld symbol is a stagger weld.

## Signature

```csharp
System.Boolean GetStagger()
```
## Parameters

None.

## Return Value

True if this is a stagger weld, false if not

## Remarks

To set the weld symbol to a stagger weld, use
IWeldSymbol::SetStagger
.
