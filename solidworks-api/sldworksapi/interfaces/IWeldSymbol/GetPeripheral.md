# IWeldSymbol.GetPeripheral

Gets whether this is a peripheral weld.

## Signature

```csharp
System.Boolean GetPeripheral()
```
## Parameters

None.

## Return Value

True if a peripheral weld, false if not

## Remarks

To set the peripheral property of a weld symbol, use
IWeldSymbol::SetPeripheral
.
