# ISketch.GetSketchContours

Gets the sketch contours in this sketch.

## Signature

```csharp
System.Object GetSketchContours()
```
## Parameters

None.

## Return Value

Array of sketch contours

## Remarks

This method works even if the sketch is suppressed.
