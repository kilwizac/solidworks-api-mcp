# ISketch.GetSketchTextSegments

Gets the sketch segments that represent the selected text in the sketch.

## Signature

```csharp
System.Object GetSketchTextSegments()
```
## Parameters

None.

## Return Value

Array of sketch segments for the selected text in this sketch
