# IDesignTable.LastError

Gets or sets the last error that occurred in this design table.

## Signature

```csharp
System.Int32 LastError {get; set;}
```
## Parameters

None.

## Return Value

Last error that occured in this design table as defined in swDesignTableErrors_e

## Remarks

Getting this property returns the last error generated by a request to modify the design table. Setting this property allows you to clear the last design table error that occurred.
