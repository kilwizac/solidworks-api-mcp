# IDesignTable.Detach

Detaches the design table from the Microsoft Excel sheet.

## Signature

```csharp
void Detach()
```
## Parameters

None.

## Return Value

Unknown.

## Remarks

Call this method after you have finished using the other
IDesignTable
methods.
This function does not specifically deactivate the design table.
