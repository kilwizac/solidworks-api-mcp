# IWizardHoleFeatureData2.NearCounterSinkDiameter

Gets or sets the diameter of the near side Hole Wizard countersink feature.

## Signature

```csharp
System.Double NearCounterSinkDiameter {get; set;}
```
## Parameters

None.

## Return Value

Near side countersink diameter

## Remarks

This property is relevant only for swCounterSunk and swCounterSunkDrilled holes.
See
Accessing Selections that Define Features
for additional details.
