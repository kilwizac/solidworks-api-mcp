# IBody2.GetProcessedBodyWithSelFace

Gets a processed body.

## Signature

```csharp
System.Object GetProcessedBodyWithSelFace()
```
## Parameters

None.

## Return Value

Body, which is a copy of the body for this part

## Remarks

This method requires that you preselect one or more faces.
