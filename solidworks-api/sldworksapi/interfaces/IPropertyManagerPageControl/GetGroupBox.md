# IPropertyManagerPageControl.GetGroupBox

Gets the PropertyManager group box to which this control belongs.

## Signature

```csharp
PropertyManagerPageGroup GetGroupBox()
```
## Parameters

None.

## Return Value

PropertyManager page group box or NULL if the control is not in a PropertyManager group box
