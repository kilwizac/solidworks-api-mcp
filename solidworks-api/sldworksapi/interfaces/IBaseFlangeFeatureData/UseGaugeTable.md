# IBaseFlangeFeatureData.UseGaugeTable

Gets or sets whether to use a gauge table for this base flange feature.

## Signature

```csharp
System.Boolean UseGaugeTable {get; set;}
```
## Parameters

None.

## Return Value

True to use a gauge table, false to not
