# IView.GetFirstCenterMark2

Gets the first center mark in the view.

## Signature

```csharp
System.Object GetFirstCenterMark2()
```
## Parameters

None.

## Return Value

First center mark

## Remarks

This method obsoletes IView::GetFirstCenterMark by supporting inactive sheets.
