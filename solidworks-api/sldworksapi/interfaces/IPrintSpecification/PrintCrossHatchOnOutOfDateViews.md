# IPrintSpecification.PrintCrossHatchOnOutOfDateViews

Gets or sets whether to print a cross hatch on out-of-date views.

## Signature

```csharp
System.Boolean PrintCrossHatchOnOutOfDateViews {get; set;}
```
## Parameters

None.

## Return Value

True to print a cross hatch on out-of-date views, false to not
