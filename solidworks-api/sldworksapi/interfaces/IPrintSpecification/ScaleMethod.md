# IPrintSpecification.ScaleMethod

Gets or sets the page selection option for printing.

## Signature

```csharp
System.Int32 ScaleMethod {get; set;}
```
## Parameters

None.

## Return Value

Page selection option as defined in swPrintSelectionScaleFactor_e

## Remarks

For details about applying a scale factor to the current drawing sheet, see the
Print Specification
Help topic in the SOLIDWORKS Help.
