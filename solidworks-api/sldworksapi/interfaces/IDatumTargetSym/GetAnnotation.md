# IDatumTargetSym.GetAnnotation

Gets the IAnnotation object for this specific datum target symbol.

## Signature

```csharp
System.Object GetAnnotation()
```
## Parameters

None.

## Return Value

Annotation

## Remarks

The
IAnnotation
object is a higher-level object that contains methods general to all types of annotations.
