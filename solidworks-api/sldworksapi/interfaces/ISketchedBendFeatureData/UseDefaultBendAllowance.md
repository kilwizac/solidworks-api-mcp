# ISketchedBendFeatureData.UseDefaultBendAllowance

Gets or sets whether to use the default bend allowance for this sketched bend.

## Signature

```csharp
System.Boolean UseDefaultBendAllowance {get; set;}
```
## Parameters

None.

## Return Value

True uses the default bend allowance, false does not
